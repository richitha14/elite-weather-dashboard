# Elite Weather Dashboard

A premium React weather dashboard with live forecasts, AQI insights, and responsive modern UI.

## Features

* Search city weather
* Current temperature
* Humidity
* Wind speed
* AQI data
* Sunrise / Sunset
* Dark / Light mode
* Responsive design

## Tech Stack

* React
* Vite
* JavaScript
* CSS
